# GigConnect – Full Stack Service Marketplace

GigConnect is a full-stack web application that allows users to discover local professionals based on skills, city, and name-based search.

---

## 🚀 Features

- Search professionals by skill, name, and location
- Dynamic filtering and real-time results
- Responsive UI
- RESTful backend APIs

---

## 🛠 Tech Stack

### Frontend
- HTML, CSS, JavaScript

### Backend
- Node.js
- Express.js
- MongoDB
